# Nx Sample

## Use this repo

Those are the important commands:

```bash
# run blog package
nx serve blog
# build blog package
nx build blog
# display dependency graph
nx graph
# show affected projects (after files were changed)
nx print-affected --select=projects
# build only affected projects
nx affected --target=build
# invoke linter
npx eslint .
```
